API限制说明


API连接限制

为了公平分配系统资源，防止API被滥用或恶意使用，开发者首先需要注册应用，才能够访问CheckNerds API。

若不注册的话，CheckNerds 也提供了开放应用API帐号可供使用，但是限制为200次/小时。

如需注册API帐号，请API用户提交所开发的应用信息(项目名称，详细介绍)到 [联系CNBorn] ，审核通过后可获得访问限制更为宽松的App访问权限。审核通过的应用会获得400次/小时的访问权限。如果开发者还需要更高的需求，可以再次提交审核，可提高到800次/小时的访问权限。

  • 测试应用API帐号（初期开放的免注册应用程序认证），200次/小时
  • 普通应用API，400次/小时
  • 特殊应用API，800次/小时

  • 用户API调用限制 400次小时
