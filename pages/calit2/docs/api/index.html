API 概述



从2008年10月发布到现在，CheckNerds始终是一个基于Web的个人事项管理程序，也就是说，用户只有通过不同设备的浏览器访问CheckNerds的Web界面（无论是普通版本还是移动设备版本），才能够查看和管理事项。

这种方式有着很强的局限性。为了让CheckNerds 成为一个可以更方便使用的工具，而不仅仅是互联网上的一个普通的网站，2009年6月，CheckNerds API (应用程序编程接口）正式公开，为开发人员和普通用户便捷地访问他们存放在CheckNerds 的信息成为了可能。

开发者可以撰写程序，直接访问 CheckNerds 服务器来访问数据。应用程序身份和用户身份验证过后，即可收到 CheckNerds 返回的数据。期间完全无需打开浏览器，访问 CheckNerds 网站。

API的开放，为 CheckNerds 的应用提供了更多的可能。它可以以软件的形式出现在各种应用平台上；也可以将 CheckNerds 的功能集成到已有的软件中，作为插件和扩展的面貌出现；甚至可以抛开 CheckNerds 的主旨——记录事项，而单单把这个API 平台作为一个 App Engine 上的数据操作接口来实现自己的云数据存储环境。这一切均可能实现，只要开发人员的创意无限，针对 CheckNerds API的应用程序就可以有任何可能。

对于普通用户来说，随着基于 CheckNerds API开发的应用程序的增多，就意味着有着更多访问 CheckNerds 的方式，最终可以让他们更加快捷、方便地使用这个工具。
